1) Problem Statement

In a local election, each candidate is assigned a unique numeric ID. After collecting the votes, you need to determine if any candidate received more than n−2 votes, where n is the total number of votes cast. This will help in identifying if there is a clear winner with significant support.

Example : 
Let's assume we've 2 [1, 4] candidates and 5 votes cast.
After the voting, the final voting is as follows [1, 4, 1, 1, 1].
As we can see, candidate 1 got more votes than n-2 (5 - 2 = 3) times, which is 4 times.
So, the output is 1.

Your task is to write a program that determines if there is a candidate ID that received more than n−2 votes using Linear search.
Input format :
The first line contains an integer n, representing the total number of votes cast.
The second line contains n space-separated integers, each representing the candidate ID for each vote.
Output format :
The output displays the following format:
If a candidate received more than n−2 votes, output their ID.
If no candidate meets this criterion, output "No majority candidate found."

Refer to the sample output for formatting specifications.
Code constraints :
1 ≤ n ≤ 10
1 ≤ candidate ID ≤ 100
Sample test cases :
Input 1 :
5
1 4 1 1 1
Output 1 :
1
Input 2 :
8
12 12 12 12 13 12 12 12
Output 2 :
12
Input 3 :
5
56 58 45 54 55
Output 3 :
No majority candidate found.

class ElectionAnalysis
{
    public static int findMajorityCandidate(List<Integer> votes,int n)
    {
        for(int i=0;i<votes.size();i++)
        {
            int v=votes.get(i);
            int c=0;
            for(int j=0;j<votes.size();j++)
            {
                if(v==votes.get(j))
                {
                    c++;
                }
            }
            if(c>n-2)
            {
                return v;
            }
        }
        return -1;
    }
}

class VersionHelper
{

    {
        int l=0,r=v.size()-1;
        int min=v.get(0);
        while(l<r)
        {
            int m=(l+r)/2;
            if(v.get(m)>v.get(r))
            {
                l=m+1;

class BinarySearchHelper
{
    public static int binarySearchCount(List<Integer> arr,int key)
    {
        int count=0;
        int l=0,h=arr.size()-1;
        while(l<=h)
        {
            int m=(l+h)/2;
            if(arr.get(m)<=key)
            {
                count=m+1;
                l=m+1;
            }
            else
            {
                h=m-1;
            }
        }
        return count;
    }

6) Problem Statement

Arsh is working on enhancing a recommendation system for a shopping application. He needs to develop a feature that suggests the closest product prices to a target price entered by a user. The list of product prices is sorted in non-decreasing order, and he wants the system to find the K closest prices to the target price. If two prices have the same difference from the target, the higher price should be preferred.

To implement this feature, Arsh needs to write a program that identifies and prints the K closest prices to the given target price from the sorted list of prices.

Example

Input:
5
2 4 6 8 10
5
3
Output:
6 4 8 

Explanation
The closest prices to 5 are determined by their absolute differences: 6 (|6-5|=1), 4 (|4-5|=1), and 8 (|8-5|=3). Since ties are resolved by choosing the higher number, the closest prices are 6, 4, and 8. Therefore, the output is 6 4 8.
Input format :
The first line contains an integer n, representing the number of product prices.
The second line contains n space-separated integers, representing the sorted product prices.
The third line contains an integer x, denoting the target price.
The fourth line contains an integer k, representing the number of closest prices to find.

The output displays k space-separated integers representing the k closest prices to the target value.

Refer to the sample output for the formatting specifications.
Code constraints :
1 ≤ n ≤ 10
1 ≤ product prices ≤ 100
Prices are given in ascending order.
1 ≤ k ≤ n
Sample test cases :
Input 1 :
5
2 4 6 8 10
5
3
Output 1 :
6 4 8 
Input 2 :
8
1 3 5 7 9 11 13 15

class PriceRecommendation
{
    public static void printKClosest(List<Integer> p, int t,int k,int n)
    {
        int l=0,r=n-1;
        while(r-l>=k)
        {
            int dl=Math.abs(p.get(l)-t);
            int dr=Math.abs(p.get(r)-t);
            if(dl>dr)
            {
                l++;
            }
            else if(dl<dr)
            {
                r--;
            }
            else
            {
                l++;
            }
        }
        List<Integer> li=new ArrayList<>();
        for(int i=l;i<=r;i++)
        {
            int s=p.get(i);
            li.add(s);
        }
        li.sort((a,b)->{
        int da=Math.abs(a-t);
        int db=Math.abs(b-t);
        if(da!=db) return da-db;
        return b-a;
       }  );
        for(int i=0;i<li.size();i++)
        {
            System.out.print(li.get(i)+" ");
        }
    }
}

    {
        int p=-1;
        for(int i=0;i<arr.size();i++)
        {
            if(x==arr.get(i))
            {
                p=i;
            }
        }
        return p;
    }
    static int findSecondLargest(List<Integer> arr)
    {
        int h=arr.get(0);
        int sh=-1;
        for(int i=0;i<arr.size();i++)
        {
            //int n=arr.get(i);

                sh=h;
                h=arr.get(i);

            {
                sh=arr.get(i);
            }
        }

    }

class SingleNonDuplicateFinder
{
    static int findSingleNonDuplicate(List<Integer> a)
    {
        int r=-1;
        HashMap<Integer,Integer> m=new HashMap<>();
        for(int i=0;i<a.size();i++)
        {
            m.put(a.get(i),m.getOrDefault(a.get(i),0)+1);
        }
        for(Map.Entry<Integer,Integer> e:m.entrySet())
        {
            if(e.getValue()==1)
            {
              r=e.getKey();  
            }
        }
        return r;
    }
}
